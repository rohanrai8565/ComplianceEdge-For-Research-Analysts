# ComplianceEdge — For Research Analysts

A clean, responsive landing page prototype for ComplianceEdge, a service that helps research analysts manage regulatory compliance.

## Features

- Modern, responsive single-page layout
- Hero section with illustration, trust and rating badges
- Feature grid outlining key offerings
- Mobile-friendly navigation and layout

## Files

- `index(1).html` — Main HTML file containing markup and embedded CSS/JS for the prototype.

## Usage

1. Open `index(1).html` in your browser to view the landing page:

```bash
# On Windows
start "" "index(1).html"

# Or open directly from your file manager / VS Code Live Server
```

## Development

- The styles are inlined inside the `<head>` of `index(1).html`. For larger projects, extract CSS to a separate file (e.g., `styles.css`).
- SVG artwork is embedded inline in the hero card; you can replace it with other illustrations or optimized assets.
- Mobile navigation toggling is handled by a small script at the bottom of the HTML file.
